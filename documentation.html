System Architecture

The system is divided into three layers: a data pipeline (Python scripts run on Google Colab), a model serving layer (FastAPI on OCI Cloud), and a web frontend (React + Vite).

# Directory structure
fyp-26-s2-27/
├── data_collection/    # API scrapers
├── preprocessing/     # Normalisation scripts
├── sentiment/         # FinBERT scoring
├── model/             # BiLSTM training
├── backend/           # FastAPI server
└── frontend/         # React + Vite app

03

FinBERT Sentiment Model

Base model: yiyanghkust/finbert-tone — a BERT model pre-trained on financial text and fine-tuned for tone classification.

Output labels

LabelCountDescription
positive2,565Bullish or favourable news sentiment
negative3,662Bearish, risk, or unfavourable sentiment
neutral1,808Factual reporting, no directional bias

Daily aggregation

# Aggregate per stock per day
daily_sentiment = df.groupby(['ticker', 'date']).agg({
  'score': 'mean',
  'label': lambda x: x.value_counts().idxmax()
}).reset_index()
04

BiLSTM Model

A Bidirectional Long Short-Term Memory network that processes sequences of 20 trading days (sliding window) to predict the next-day direction.

ParameterValue
ArchitectureBidirectional LSTM
Input featuresOHLCV + RSI + MACD + BB + sentiment score
Sequence length20 trading days
OutputBinary — UP (1) / DOWN (0)
Train/test splitChronological (no data leakage)
FrameworkPyTorch
Training environmentGoogle Colab (GPU)

The backend exposes a REST API consumed by the React frontend.

# Returns prediction + sentiment for a given ticker

# Returns aggregated daily FinBERT scores

# Health check endpoint

06

Setup & Run

Backend (FastAPI)

# Install dependencies
pip install -r backend/requirements.txt

# Start the server
cd backend
uvicorn main:app --host 0.0.0.0 --port 8000

Frontend (React + Vite)

# Install & run dev server
cd frontend
npm install
npm run dev

# Build for production
npm run build

Model training (Google Colab)

# Run in order:
1. 01_data_normalisation.ipynb
2. 02_finbert_scoring.ipynb
3. 03_bilstm_training.ipynb
