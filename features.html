What the system does

Key capabilities across the data pipeline, machine learning models, and web application interface.

01 / Data Pipeline

Multi-source financial data collection

Automated scripts pull historical OHLCV price data and news headlines from five public APIs, deduplicate and clean the records, then store them in a unified format ready for feature engineering.

02 / NLP

FinBERT sentiment scoring

Every financial news headline is passed through a FinBERT transformer model — a BERT variant fine-tuned on financial text — to produce Positive, Negative, or Neutral sentiment labels with confidence scores. Scores are aggregated daily per stock.

03 / Deep Learning

Bidirectional LSTM prediction model

A BiLSTM neural network processes rolling windows of combined price and sentiment features. The bidirectional architecture captures both past and contextual temporal patterns, outputting a probability for the next-day UP / DOWN direction.

04 / Web Application

Live prediction interface

A React frontend backed by a FastAPI Python server. Users select a stock from a dropdown, the system auto-triggers a BiLSTM inference call, and returns the directional prediction alongside the FinBERT sentiment gauge — making the model's inputs visible to the user.
