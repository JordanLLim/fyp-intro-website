System Features

What the system does

Key capabilities across the data pipeline, machine learning models, and web application interface.

01 / AI Prediction

Multi-source financial data collection

Automated scripts pull historical OHLCV price data and news headlines from five public APIs, deduplicate and clean the records, then store them in a unified format ready for feature engineering.

02 / Real-time Stock Data

FinBERT sentiment scoring

Every financial news headline is passed through a FinBERT transformer model to produce Positive, Negative, or Neutral sentiment labels with confidence scores. Scores are aggregated daily per stock.

03 / Education Resources

XGBoost prediction model

A XGBoost model processes rolling windows of combined price and sentiment features. Thevarchitecture captures both past and contextual temporal patterns, outputting a probability for the next-day UP / DOWN direction.

PyTorchXGBoostSliding windowChronological split

features = [
  'open', 'high', 'low', 'close',
  'volume', 'rsi', 'macd',
  'bb_upper', 'bb_lower',
  'sentiment_score' # FinBERT
]

04 / Professional consultant

Live prediction interface

A React frontend backed by a FastAPI Python server. Users select a stock, the system auto-triggers XGBoost inference, and returns the directional prediction alongside the FinBERT sentiment gauge.
