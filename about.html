About the Project

Webservice for Stock Market Prediction

An AI-driven financial intelligence platform that transforms complex market data and news sentiment into clear, actionable investment insights — for everyone from beginner investors to seasoned traders.

Background

Problem & Approach

The Problem

Individual investors face information overload — countless real-time news, price data, and technical signals that are difficult to interpret without professional knowledge.

Most existing platforms focus on powerful analytics but sacrifice usability, leaving beginner investors overwhelmed. Others lack sentiment analysis, missing the "why" behind price movements.

Our Approach

A full-stack web platform combining XGBoost + FinBERT AI predictions with real-time dashboards, simulated portfolios, expert portfolios, and a community forum.

The platform supports five user roles — Basic User, Premium User, Consultant, Administrator, and Guest — each with tailored access to features matching their needs.

Vision

Our vision

We are dedicated to transforming how individuals understand and interact with financial markets. Our vision is a platform where data-driven insights are at the core of every investment decision — accessible to anyone with an internet connection, not just professionals.

For Beginner Investors

Predict trends with sentiment intelligence, monitor markets in real-time, simplify complex patterns through interactive dashboards, and learn through a Knowledge Hub and expert portfolios.

For All Users

Empower smarter investing with real-time news insights, enhance analysis with interactive charts, and improve trading decisions through direct interaction with certified stock consultants.

Architecture

System architecture

Three-tier architecture connecting a React.js frontend, FastAPI backend, and MySQL database — with an external stock API integration and ML prediction engine.

LayerDescription
Presentation LayerReact.js frontend — users search stocks, view dashboards, manage watchlists, check predictions, and access account features via an interactive web interface.
Application LayerFastAPI backend — handles user authentication, role-based access, stock data processing, AI prediction requests, portfolio functions, and all system logic.
Data LayerMySQL database — stores user accounts, roles, watchlists, portfolio records, transaction history, prediction results, alerts, forum posts, and expert content.
