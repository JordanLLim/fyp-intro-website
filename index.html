Overview

What we built

A full-stack stock market prediction web platform supporting multiple user roles, from beginner investors to professional consultants and system administrators.

Problem

Individual investors face information overload and struggle to interpret real-time market data, news sentiment, and technical indicators simultaneously — leading to uninformed trading decisions.

Solution

An intelligent web platform that fuses XGBoost + FinBERT AI predictions with real-time dashboards, simulated portfolios, expert consultation, and a community forum — all in one accessible interface.

Who It's For

Beginner to experienced investors (primarily the 20–30 age group), financial consultants who want to share curated portfolios, and platform administrators managing user access and content.

Tech Stack

React.js frontend · FastAPI backend · MySQL database · XGBoost + FinBERT ML pipeline · Alpha Vantage API · Cloud hosting — built with a three-tier architecture for scalability.
