Overview

What we built

An end-to-end prediction system bridging financial data engineering, deep learning, and a live web interface.

Problem

Traditional price models ignore news. Financial sentiment carries real signal — earnings surprises, regulatory changes, and market narratives all move stocks in ways that historical prices alone cannot predict.

Solution

A hybrid BiLSTM + FinBERT pipeline that fuses OHLCV technical features with daily sentiment scores to produce next-day UP / DOWN directional forecasts per stock.

Data

Over 8,400 financial news articles and 12,500 trading days across 10 large-cap NASDAQ tech stocks — sourced from five APIs including yfinance, Finnhub, and NewsAPI.

Outcome

A fully deployed web application with live prediction, a real-time sentiment gauge, and a reproducible Colab training pipeline demonstrating research rigour in a real financial domain.
