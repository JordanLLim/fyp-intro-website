How it works

Using the prediction interface

01
Select a stock
Choose any of the 10 tracked tickers from the dropdown — AAPL, MSFT, GOOGL, AMZN, NVDA, META, TSLA, AVGO, ORCL, or AMD.
02
Auto-trigger inference
The system automatically calls the FastAPI backend, which runs the BiLSTM model on the latest normalised price and sentiment features.
03
View results
The prediction card shows the UP / DOWN directional forecast, the model confidence score, and the FinBERT sentiment gauge for that stock's recent news.
